# Sotero Sampler Suite

**Sotero Sampler Suite** é uma plataforma exclusiva de criação e reprodução de bibliotecas de áudio desenvolvida pela **SoteropolySamples**.

## Componentes da Suite

- **SoteroBuilder**: Ferramenta standalone para criação, mapeamento e exportação de bibliotecas no formato proprietário `.spsa`.
- **SamplerPlayer**: Plugin (VST3/AU) e app standalone otimizado para a reprodução fiel das bibliotecas Sotero.
- **Common**: Biblioteca compartilhada contendo o motor de compressão, criptografia e especificações do formato.

## Começando

### Para Usuários
1. Abra o **SamplerPlayer**.
2. Vá em `SETUP` > `LOAD SOTERO LIBRARY`.
3. Selecione um arquivo `.spsa` para carregar os samples.

### Para Desenvolvedores
Consulte o [Developer Guide](Docs/DEVELOPER_GUIDE.md) para instruções de configuração de ambiente e padrões de código.
